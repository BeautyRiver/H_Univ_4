# 독일의 문화와 예술 - 3주차 시험 족보
> 주제: **독일의 교육** | 독일어 발음 표기 추가 버전

---

## 1. 핵심 주제 정리 (중요도 순)

### ① 조기 진로 결정과 중등 교육 체계
초등학교(4년) 졸업 후 **4학년 때** 진로를 조기 결정.  
세 갈래로 나뉨 → 하웁트슐레 / 레알슐레 / 김나지움

### ② 대학의 평준화와 공공성
- 대부분 국립, **대학 서열 없음**
- 등록금 거의 없음
- 대학 진학 필수 조건: **아비투어** 성적

### ③ 전인교육(빌둥) 지향
빌헬름 폰 훔볼트 이념 → 직업 교육보다 **순수 학문 + 교양** 중심  
직업 교육은 **아우스빌둥**으로 별도 운영

### ④ 교육 분권화
연방정부 아닌 **16개 주 정부**에 교육 권한 위임 → 주마다 교육과정 다름

### ⑤ 최근 변화
김나지움·대학 진학률 급상승 (37% → **약 55.5%**) → 전문 기술 인력 부족 우려

## 2. 핵심 용어 + 독일어 발음

| 한국어 | 독일어 | 발음 | 설명 |
|--------|--------|------|------|
| 유치원 | Kindergarten | **킨더가르텐** | 놀이 중심 보육 시설. 영어 Kindergarten의 어원 |
| 초등학교 | Grundschule | **그룬트슐레** | 4년제 의무교육. 4학년 때 진로 결정 |
| 기본학교 | Hauptschule | **하웁트슐레** | 수공업·기술직 양성. 9학년 졸업 |
| 실업/전문고 | Realschule | **레알슐레** | 기술직·사무직 양성. 10학년 졸업 |
| 인문계 고교 | Gymnasium | **김나지움** | 대학 진학 목적. 12~13학년 졸업 |
| 통합학교 | Gesamtschule | **게잠트슐레** | 위 세 학교를 통합. 조기 결정 부담 완화 |
| 대학입학자격시험 | Abitur | **아비투어** | 독일판 수능. 김나지움 졸업 시 응시 |
| 직업훈련과정 | Ausbildung | **아우스빌둥** | 학교 이론 + 산업체 실무 병행 이원화 교육 |
| 전인교육 | Bildung | **빌둥** | 인격 완성을 위한 교양 교육. 괴테 교양소설과 연관 |
| 학생식당 | Mensa | **멘자** | 대학 구내식당 |
| 성적 | Note | **노테** | 1점이 최고점. 숫자 낮을수록 우수 |
| 이공계 학위 | Diplom | **디플롬** | 자연과학·기술·경영 전공 전통 학위 |
| 인문계 학위 | Magister | **마기스터** | 철학·역사 등 인문학 전통 학위 |
| 국가고시 | Staatsexamen | **슈타츠엑사멘** | 의사·교사 등 전문직 자격 시험 |

> 💡 **발음 주의**  
> - Gymnasium → **짐**나지움 아니라 **김**나지움  
> - Gesamtschule → 게**잠**트슐레 (t 약하게)  
> - Abitur → 아비**투어** (영어식 "아비터" 아님)  
> - Mensa → **멘**자 (멘**사** 아님)

## 3. 인물

**빌헬름 폰 훔볼트 (Wilhelm von Humboldt / 빌헬름 폰 훔볼트)**
- 베를린 훔볼트 대학교 설립자
- 주장: 대학 교육 = 순수 학문 + 자율권 + 전인교육(빌둥)
- 직업 교육은 대학 밖에서 따로

## 4. 숫자·연도 암기

| 숫자 | 의미 |
|------|------|
| **4년** | 그룬트슐레(초등) 학업 기간 |
| **4학년** | 진로 조기 결정 시기 |
| **16개** | 독립적 교육과정 운영하는 연방 주 수 |
| **10%** | 사립학교 비율 (나머지 90%+ 공립·무상) |
| **1점** | 노테(Note) 최고점 |
| **37% → 55.5%** | 대학 진학률 변화 (2005년 → 최근) |
| **9월** | 초중고 새 학기 시작 |
| **10월** | 대학 겨울학기(메인) 시작 |

5. 입학은 3월이다

**Q2.** 대학 진학을 목표로 하는 독일 인문계 고등학교는?
1. 그룬트슐레
2. 하웁트슐레

**Q3.** 독일 대학 교육의 특징으로 옳지 **않은** 것은?
1. 대부분 국립이다
2. **엄격한 대학 서열이 존재한다** ✅ (← 틀린 것)
3. 학생에게 교통비·의료보험 혜택이 있다
4. 교수는 국가 공무원이다
5. 순수 학문·전인교육을 중시한다

---

**Q4.** 독일 학위·평가 제도 설명으로 옳은 것은?
1. 숫자 높을수록 우수한 성적이다
2. 수능 이름은 바칼로레아다
3. 인문학 전공 학위는 디플롬이다
4. **2002년 이후 학사/석사 분리 제도 도입** ✅
5. 아비투어만 통과하면 의사·교사 자격이 주어진다

---

**Q5.** 독일 교육의 최근 변화로 알맞은 것은?
1. 김나지움 진학률이 하락하고 있다
2. **대학 진학률이 37%에서 55% 이상으로 상승했다** ✅
3. 미국식 무한 경쟁을 전면 도입했다
4. 연방정부가 교육권을 100% 회수했다
5. 아우스빌둥을 폐지했다
