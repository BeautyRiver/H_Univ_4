# 1주차
## 결혼의 의미와 변화

---

### 1. 핵심 개념

| 용어 | 설명 |
|------|------|
| **결혼의 의미** | 단순한 만남을 넘어 육체적·정서적·경제적·사회적·법적인 결합이자 서로 간의 공약(약속) |
| **일상가사대리권** | 부부 중 한 사람이 일상적인 가사(생활비 대출, 자녀 교육비 등)에 대해 법적·행정적 결정을 내릴 때, 부부 공통의 의사로 인정하여 연대 책임을 지게 하는 권리 |
| **결혼을 통한 성인 신분 획득** | 만 18세 이상이면 부모 동의하에 결혼 가능. 만 19세 미만이라도 결혼하면 법적 의사결정을 스스로 할 수 있는 성인으로 인정 |
| **만혼화** | 전체 혼인 건수가 줄어드는 가운데, 남녀 모두 초혼 연령이 점차 높아지는 현상 |
| **비혼 vs 미혼** | 미혼 = '아직' 결혼 안 한 상태 / 비혼 = 본인의 의지로 결혼 '하지 않겠다'고 결정한 상태 |
| **사실혼** | 실질적으로 부부로서 공동 생활을 하고 주변에서도 인정받지만, 법적으로 혼인신고만 하지 않은 상태. 이혼 시 재산 분할 등 일부 권리 인정됨 |
| **팍스 (PACS / 시민연대계약)** | 1999년 프랑스 도입. 동거 커플(이성/동성 모두)이 같은 주소지에 등록하면 결혼과 유사한 사회복지 혜택 제공. 단, 해지 간단 / 위자료·자녀 양육비 청구 의무 없음 / 상속권 없음 |

---

### 2. 중요 학자

**브로드릭 (Broderick)**
결혼의 9가지 특징을 열거.
- 가장 기본적인 사회적 단위이자 인구학적 사건
- 새로운 친족(인척) 네트워크를 형성하는 법적 계약
- 부부의 경제적 결합 / 합법적 성적 활동의 환경
- 노동력 재생산 및 자녀 사회화 단위

---

### 3. 통계 및 수치

| 항목 | 수치 |
|------|------|
| **2022년 초혼 연령** | 남성 33.7세 / 여성 31.3세 / 연령차 2.5세 |
| **결혼 안 하는 이유 1순위** | 남성: 결혼 자금 부족 / 여성: 결혼의 필요성을 느끼지 못해서 |
| **결혼 안 하는 이유 3순위** | 남성: 고용 불안정 / 여성: 출산·양육 부담 |
| **30대 미혼 비율** | 1990년 평균 6.8% → 2020년 42.5% (남성은 50% 이상) |
| **청년층 비혼 동거 동의율** | 80.9% (여성이 남성보다 동의 비율 약간 낮음) |
| **동거의 최대 장점** | 정서적 유대감 및 안정감 형성 (약 90%) |
| **동거의 단점** | 주변 부정적 시선 / 법적 보호자 미인정 / 사회복지 혜택 배제 |

---

### 4. OX 문제

**Q1.** 결혼을 하면 법적으로 8촌 이내 혈족과 4촌 이내 인척을 포함하는 친족 관계가 형성된다.
> ✅ **O** — 민법에 따라 결혼 시 배우자의 혈족 등이 포함된 새로운 친족(인척) 네트워크 형성

**Q2.** 혼전임신·동정심·억압적 가정환경으로부터의 도피는 결혼의 일반적이고 긍정적인 동기이다.
> ❌ **X** — 이혼 등 2차 문제를 유발할 확률 높은 **부정적 동기**

**Q3.** 만 18세 미성년자가 부모 동의하에 결혼하더라도 만 19세 전까지는 주요 계약 시 부모 동의가 필요하다.
> ❌ **X** — 결혼과 동시에 법적 의사결정을 스스로 할 수 있는 성인으로 인정됨

**Q4.** 사랑하는 사이라면 서로 자연스럽게 맞춰지므로 차이를 극복하기 위한 노력은 크게 필요하지 않다.
> ❌ **X** — 사랑하더라도 서로 다른 점이 많기 때문에 차이를 극복하려는 노력이 필수적

**Q5.** 배우자가 나의 모든 욕구를 충족시켜 줄 것이라는 기대는 현실적이다.
> ❌ **X** — 배우자를 과도하게 이상화하는 **잘못된 통념**

**Q6.** 성평등 의식 확산으로 남성 연상 커플은 줄고 여성 연상 커플이 증가하고 있다.
> ✅ **O**

**Q7.** 결혼 안 하는 이유 1순위는 남녀 모두 동일하게 '결혼 자금 부족'이다.
> ❌ **X** — 남성 1순위: 결혼 자금 부족 / 여성 1순위: 결혼의 필요성 못 느낌

**Q8.** 청년층 비혼 동거 동의율은 약 80.9%로 매우 긍정적으로 변화하였다.
> ✅ **O**

**Q9.** 사실혼은 혼인신고를 하지 않았기 때문에 재산 분할 청구 권리가 전혀 없다.
> ❌ **X** — 사실혼으로 인정받으면 재산 분할·채무 분할 청구권 인정됨

**Q10.** 팍스(PACS)는 위자료·자녀 양육비 청구 의무와 상속권이 부여된다는 점에서 법적 결혼과 동일하다.
> ❌ **X** — 팍스는 해지 간단, 위자료·양육비 청구 의무 없음, **상속권 없음** → 법적 결혼과 다름

# 2주차
## 가족의 의미와 변화

---

### 1. 핵심 개념

| 용어 | 설명 |
|------|------|
| **생활 공동체 (Familia)** | 고대(농경) 사회에서의 가족 의미. 생존을 위해 한 울타리 안에서 함께 노동하고 나누어 먹는 협력 단위 |
| **정상가족 이데올로기** | 산업사회 이후 '엄마, 아빠, 자녀'로 구성된 형태만을 '정상'으로 보고 그 외를 비정상·결손가족으로 치부하는 잘못된 편견 |
| **우애적 가족 가치** | 현대 사회의 가족 가치관. 제도보다 구성원 간의 관계·친밀감·행복을 더 중요시하는 것 |
| **가족센터 (구 건강가정지원센터)** | **사전 예방** 차원의 보편적 가족 복지 기관. 가족 여가 문화, 부모/결혼 준비 교육, 무료 상담, 아이돌보미 등 제공 |
| **건강가정의 핵심 5요소** | 친밀감(응집력) / 의사소통 / 적응력(융통성) / 지역사회와 원만한 관계 / 경제적 안정 |
| **딩크족 (DINK)** | Double Income, No Kids. 맞벌이하면서 자발적으로 자녀를 갖지 않고 부부만의 자아실현과 친밀한 생활에 집중하는 가족 |
| **출생통보제** | 병원 의료진이 국가에 아이 출생을 의무적으로 알리는 제도 (2024년 시행) |
| **보호출산제** | 미혼모 등이 신원 노출 없이 익명으로 아이를 낳도록 보호하는 제도 (2024년 시행) |
| **분거 가족** | 자녀 학업(기러기 가족)이나 직장 문제로 떨어져 지내는 가족. 자유롭고 목표 지향적이나 정서적 소외감·관계 단절 위험 |
| **비혈연 공동체 가족** | 혼인·혈연관계 없이도 생활공간을 공유하며 서로 돕는 관계. 분자가족·간헐적 가족(은혜공동체) 등 |

---

### 2. 중요 학자 및 이론

**파슨스 (Talcott Parsons) — 구조기능주의**
- 남성: 돈을 버는 **도구적 역할**
- 여성: 양육과 안식을 제공하는 **정서적 역할**
→ 성별 분업이 합리적이라고 주장 (현재는 비판받는 이론)

**올슨 (Olson) — 가족 순환 모델**
- 건강한 가족 = **응집력(친밀감)**과 **적응성**이라는 두 축에서 극단적으로 밀착되거나 분리되지 않은 **'가운데 위치(중용)'**
- 극단적 밀착도, 완전한 분리도 건강하지 않음

**황두영 작가 — 생활동반자법**
- 성인 두 명이 나라에 등록하고 서로 돌보겠다고 약속하면 가족으로서의 법적 권리(수술 동의·재산 분할·가정폭력 보호 등) 부여 제안
- 프랑스 팍스(PACS)를 모티브로 함
- 1인 가구·노인층의 고독·돌봄 문제 해결 목적

---

### 3. 통계 및 수치

| 항목 | 수치 |
|------|------|
| **가족 해체 경제적 위기** | 1998년 IMF / 2003년 카드 사태 / 2008년 글로벌 금융위기 → 조이혼율 급상승 |
| **건강가정기본법 제정** | 2004년 |
| **맞벌이 부부 무자녀 비율** | 50% 이상 (외벌이보다 높음) |
| **난임 확률** | 만 35세 이상 여성의 경우 35%까지 상승 / 남녀 원인 비율 반반 |
| **1인 가구 비율 (2022년)** | **34.5%** — 세 가구 중 한 가구 |
| **1인 가구 최대 어려움** | 경제적 문제보다 **정서적 외로움** 1위 |
| **비혈연 가족 인정 동의율** | 전 국민 **68.5%** |

### 4. OX 문제

**Q1.** 파슨스의 구조기능주의는 남성이 정서적 역할을, 여성이 도구적 역할을 맡아야 한다고 주장했다.
> ❌ **X** — 반대. 남성 = **도구적** 역할, 여성 = **정서적** 역할

**Q2.** 올슨의 가족 순환 모델에서 건강가정은 친밀감이 가장 높아 밀착된 상태를 유지하는 가족이다.
> ❌ **X** — 극단적 밀착은 건강하지 않음. **프라이버시와 융통성이 유지되는 중용**이 건강한 상태

**Q3.** 건강가정기본법은 문제 발생 후 개입하는 사후 처방 중심에서 '사전 예방' 차원의 복지로 패러다임을 바꾼 것이다.
> ✅ **O**

**Q4.** 최근 통계에서 무자녀 비율은 외벌이 부부보다 맞벌이 부부가 더 높다.
> ✅ **O** — 자아실현 욕구, 시간 한계 등으로 맞벌이 부부의 무자녀 비율이 50% 이상으로 더 높음

**Q5.** '엄마, 아빠, 자녀'가 모두 있는 가족만 정상이고 그렇지 않으면 결손 가족으로 취급하는 편견을 '정상가족 이데올로기'라 한다.
> ✅ **O**

**Q6.** 미혼부·미혼모 가족이 가장 호소하는 핵심 어려움은 '단절과 고립'이라는 심리적 문제이다.
> ✅ **O**

**Q7.** 한부모 가족은 70% 이상이 부자가구(아빠+자녀)이다.
> ❌ **X** — 대다수는 **모자가구(엄마+자녀)**

**Q8.** 비밀 입양은 자녀의 정체성 확립에 긍정적인 영향을 미친다.
> ❌ **X** — 비밀 입양은 사실이 밝혀졌을 때 배신감과 정체성 혼란 초래. 현재는 **공개 입양**이 원칙

**Q9.** 2022년 기준 가장 높은 비율의 가구 형태는 1인 가구이며, 이들은 정서적 외로움을 가장 힘든 점으로 꼽는다.
> ✅ **O** — 1인 가구 34.5%

**Q10.** 생활동반자법은 혈연이 아니더라도 함께 거주하며 돌보는 두 성인에게 법적 권리와 사회복지 혜택을 부여하자는 취지의 법이다.
> ✅ **O**

### 5. 단답형 문제

**Q1.** 맞벌이하며 자발적으로 자녀를 갖지 않는 부부를 일컫는 용어는?

**Q2.** 경제 위기로 가족 해체를 겪으며 사전 예방 차원의 복지 실현을 위해 2004년 제정된 법률은?

**Q5.** 엄마·아빠·자녀로 이루어진 구성원 형태만을 정상으로 보는 이데올로기는?
> **정상가족 이데올로기**

# 3주차
## 건강한 연대와 진정한 자립

---

### 1. 핵심 개념

| 용어 | 설명 |
|------|------|
| **느슨한 연대** | 연결의 장점은 취하되 관계에서 오는 부담은 덜어내는 현대 개인주의자들이 선호하는 연대 방법 |
| **캥거루족** | 능력이 안 되어 부모에게 계속 의존하는 청년 |
| **신캥거루족** | 취업하여 경제적 능력이 있음에도 부모와 동거하며 기능적으로 독립하지 못한 청년 |
| **자아분화 (Differentiation of Self)** | 보웬의 다세대 가족 치료 이론 핵심 개념. 원가족의 정서적 융합에서 벗어나 자율적으로 기능하는 능력 |
| **삼각관계 (Triangulation)** | 자아분화가 안 된 밀착된 부부가 갈등 시 불안 해소를 위해 가장 약한 자녀를 끌어들여 '희생양'으로 만드는 현상 |
| **니트(NEET)족** | 학교 졸업 후 취업도 안 하고 훈련도 받지 않는 무기력한 청년층 |
| **히키코모리 (은둔형 외톨이)** | 취업 실패·인간관계 상처 등으로 집이나 방에 틀어박혀 사회와 단절된 사람들 |
| **8050 문제** | 히키코모리 자녀가 50대가 되고 부모가 80대가 되어 노부모가 자녀를 부양해야 하는 심각한 상황 |
| **성인 진입기 (Emerging Adulthood)** | 10대 후반~20대 후반. 아직 정체성이 확립되지 않았지만 낙관적으로 실험하는 심리사회적 유예기 |

### 2. 중요 학자 및 이론

**애덤 스미스 (Adam Smith) — 국부론**
개인주의 ≠ 이기주의.
타인에게 역할을 떠맡기는 '의존'이 아니라 서로가 책임감을 갖고 협업하며 **'의지'**할 때 진정한 건강한 연대 가능.

**에릭슨 (Erikson) — 심리사회적 발달 단계**
- 청년기(6단계) 핵심 발달 과업: **친밀감 형성 (친밀감 대 고립감)**
- 필수 선행 조건: 청소년기의 **자아정체감** 확립

**마샤 (Marcia) — 자아정체감 4범주**

| 범주 | 위기감(탐색) | 몰입 | 설명 |
|------|------------|------|------|
| **정체성 성취** | O | O | 가장 건강한 상태 |
| **정체성 유예** | O | X | 아직 결정 못 내림. 많은 대학생 해당 |
| **정체성 유실** | X | O | 부모가 정해준 정체성을 그대로 수용 |
| **정체성 혼미** | X | X | 위기감도 몰입도 없는 가장 미성숙한 상태 |

**보웬 (Bowen) — 다세대 가족 치료 이론**
- 부모의 **미분화(낮은 자아분화)** 상태가 자녀에게 **세대 전수**됨
- 자아분화가 잘 된 사람 = 감정과 사고를 분리하여 객관적으로 판단 가능

**사랑에 관한 노르딕 이론 (핀란드 등 북유럽)**
국가가 개인을 가족·타인의 의존에서 완벽히 경제적으로 독립시켜줄 때, 동등한 개인들 사이에서 진정한 사랑과 우정이 가능하다는 철학.

# 4주차
## 사랑과 친밀한 관계 형성

---

### 1. 핵심 개념

| 용어 | 설명 |
|------|------|
| **라포 (Rapport)** | 처음 만나는 사람과 동질감(유사점)을 찾아 공감하며 형성하는 친밀감·신뢰감 |
| **자기 개방** | 라포 형성 이후, 정보의 양과 수위를 조절하며 자신에 대한 정보를 솔직하게 알려주는 과정 |
| **사랑 중독 (매니아)** | 파트너가 없으면 견디지 못하고 상대방을 과도하게 소유·집착·의존하는 병적인 사랑 상태 |
| **연애 휴지기** | 이별 후 섣불리 다른 사람을 만나 잊으려 하기보다 나에 대해 성찰하며 치유할 시간을 갖는 기간 |
| **합류적인 사랑 (Confluent Love)** | 앤서니 기든스의 개념. 나의 고유한 독립성(정체성)을 유지하면서 유대감을 형성해 우리의 새로운 정체성을 발전시키는 현대적 사랑 |

---

### 2. 중요 학자 및 이론

**에리히 프롬 (Erich Fromm)**
- 사랑 = 수동적 감정이 아니라 **능동적인 의지와 노력(기술)**의 산물
- 사랑의 4요소: **보호, 책임, 존경, 지식**
- ⚠️ 지식 없는 보호 = 맹목과 집착 / 존경 없는 책임 = 지배와 소유

**벨 훅스 (bell hooks)**
- 단순한 설렘의 감정을 경계
- 사랑 = **영혼과 영혼이 연결**되어 서로의 영적 성장과 자아를 확장시키는 의지

**레이스 (Reiss) — 사랑의 수레바퀴 이론**
```
라포 형성 → 자기 노출 → 상호 의존 → 친밀감 욕구 충족
```
순환 발전하지만, 노력하지 않으면 가역적으로 되돌아갈 수 있음

**스턴버그 (Sternberg) — 사랑의 삼각형 이론**
3요소: **친밀감 + 열정 + 헌신**

| 유형 | 구성 | 사례 |
|------|------|------|
| **도취적 사랑** | 열정만 | 짝사랑, 연예인 팬의 사랑 |
| **공허한 사랑** | 헌신만 | 정략결혼 |
| **낭만적 사랑** | 친밀감+열정 | 대학가 연애, 첫사랑 |
| **우애적 사랑** | 친밀감+헌신 | 열정 식은 중년 부부 |
| **완전한 사랑** | 친밀감+열정+헌신 | 3요소가 균형 잡힌 이상적 사랑 |

> ⚠️ 열정은 초기에 급상승 후 **길어야 3년(보통 18~30개월)** 이내에 하강

**존 리 (John Lee) — 사랑의 색채 이론**

| 색 | 유형 | 특징 |
|----|------|------|
| **원색 1** | 에로스 (Eros) | 육체적·열정적 사랑 |
| **원색 2** | 루더스 (Ludus) | 밀당·유희적·이기적 사랑 |
| **원색 3** | 스토르게 (Storge) | 편안하고 은근한 우정 같은 사랑 |
| **혼합 1** | 매니아 (에로스+루더스) | 집착·소유적 사랑 |
| **혼합 2** | 프라그마 (루더스+스토르게) | 조건을 따지는 논리적·실용적 사랑 |
| **혼합 3** | 아가페 (에로스+스토르게) | 무조건적·헌신적·희생적 사랑 |

---

### 3. 통계 및 수치

| 항목 | 수치 |
|------|------|
| **열정의 유효 기간** | 보통 18~30개월. 길어야 **3년 이내** 하강 |
| **이별 상실의 단계 회복 기준** | **6개월** 넘어도 일상 회복 안 되면 전문가 도움 필요 |
| **콩깍지 호르몬** | 도파민, **페닐에틸아민** (초콜릿 성분) |

|------|------|
| **동질성의 원리 (유사성의 원리)** | 나이·학력·경제적 수준·종교·가치관 등 자신과 비슷한 사람끼리 커플이 될 확률이 높다는 원리 |
| **보상성의 원리 (상보성의 원리)** | 서로 다른 점이 상호 보완되어 매력을 느끼고 커플로 이어지는 원리 (예: 주도적인 사람 + 순종적인 사람) |
| **교환자원 공평성의 원리** | 관계가 유지되려면 서로 주고받는 자원이 공평해야 함. 자원 = 물질적 자원 + 위로·지지 등 내적·정서적 자원 모두 포함 |
| **결혼의 운전면허증 (마음의 혼수)** | 물질적 혼수보다 사고 없이 목적지에 도달하기 위한 준비. 건강한 가족 관계 자질 + 미래 비전 공유 + **긍정적 의사소통 및 갈등 해결 기술(실기)** |
| **작은 결혼식(스몰 웨딩)의 변질** | 허례허식을 줄이자는 본래 취지에서 벗어나, '나만의 특별한 결혼식'을 추구하다 오히려 비용이 더 드는 형태로 변질 |

---

### 2. 중요 학자 및 이론

**우드리 (Udry) — 여과망 이론**
데이트 가능한 모든 대상에서 6단계 여과망을 거쳐 최종 배우자 결정

| 단계 | 여과망 |
|------|--------|
| 1 | **근접성** — 물리적 만남의 기회 ← 1단계임에 주의 |
| 2 | **매력** — 시각적·외적 매력 |
| 3 | **사회적 배경** — 학력·종교 등 배경의 유사성 |
| 4 | **의견 합치** — 가치관·태도의 유사성 |
| 5 | **상호보완성** — 서로 바라는 역할 충족 여부 탐색 |
| 6 | **결혼 준비** — 모든 조건 충족 시 결혼 결심 |

**머스테인 (Murstein) — SVR 이론**
관계 발전 단계에 따라 중요한 요소가 달라짐

| 단계 | 핵심 요소 | 설명 |
|------|----------|------|
| 초기 | **S (자극, Stimulus)** | 외모 등 외적 매력 |
| 중기 | **V (가치, Value)** | 가치관·인생관 동질성 탐색 |
| 후기 | **R (역할, Role)** | 상대가 내가 원하는 역할 충족하는지(상호보완성) |

**르이스 (Lewis) — 배우자 관계 형성 이론**
```
유사성 지각 → 라포 형성 → 자기 개방 → 역할 탐색 → 역할 적합성 지각 → 부부관계 형성
```

# 6주차
## 친밀한 폭력

---

### 1. 핵심 개념

| 용어 | 설명 |
|------|------|
| **교제 폭력 (데이트 폭력)** | 현재 사귀는 과정, 연인으로 발전할 가능성이 있는 관계, 연인 관계가 끝난 이후까지 포괄하는 모든 폭력. 신체적·성적·언어적·정서적·경제적 폭력, 통제(옷차림 감시 등), 스토킹 모두 포함 |
| **디지털 성범죄 / 리벤지 포르노** | 동의 없이 성적인 사진·영상을 촬영·유포하거나 이를 빌미로 협박하는 행위 |
| **가스라이팅 (Gaslighting)** | 가해자가 폭력을 '사랑'으로 포장하며 피해자를 고립시킨 뒤, 피해자가 스스로를 의심하고 자책하게 만드는 심리적 지배 행위 |
| **학습된 무기력 (Learned Helplessness)** | 반복적인 폭력과 가스라이팅에 노출된 피해자가 저항하거나 벗어날 의지조차 상실하게 되는 현상 |
| **방임** | 가정 폭력(아동·노인 학대)의 한 유형. 보호자가 유기하거나 동의 없이 시설에 강제 입소 후 연락을 끊고 방치하는 행위 |
| **조기 성숙 (가짜 성숙)** | 가정 폭력 환경에서 자녀가 정상적인 발달 과업을 거치지 못하고 부모를 대신해 어른 역할을 억지로 수행하며 겉보기에 성숙한 척 행동하게 되는 부작용 |

---

### 2. 중요 이론

**사회학습 이론 (Social Learning Theory)**
가해자가 어렸을 때 가정 폭력에 노출된 경우, 그 폭력적 방식을 모방·학습하여 성인이 된 후 감정 통제가 안 될 때 폭력 행사 가능성이 높음.
→ **폭력의 세대 전수 현상** 설명

**젠더 감수성 및 인권 감수성**
다른 성별의 입장·사상을 이해하는 젠더 감수성과 사람의 기본적 권리를 존중하는 인권 감수성이 낮을 때 교제 폭력·가정 폭력이 빈번하게 발생.

---

### 3. 통계·수치·관련 법률

**교제 폭력 통계**

| 항목 | 수치 |
|------|------|
| 피해자 성별 | 약 **80%가 여성** |
| 폭력 발생 시기 | **교제 1년 이내 약 70%** / **6개월 이내 50%** — 초반부에 집중 |
| 가해자 재범률 | **60%** (매우 높음) |
| 피해자가 가해자와 결혼 선택 비율 | 약 **40%** |

**아동 학대 통계**
- 가해자 약 **65%는 친부모** (언론 보도와 달리 계모·계부가 1위 아님)
- '내 자식이니 내 마음대로 훈육'이라는 잘못된 편견이 주요 원인

**노인 학대 통계**
- 정서적 학대 가장 많음
- 가해자 **1순위: 아들** (며느리 아님)
- 수치심·가족 보호 이유로 경찰 신고 비율 **0%에 수렴**

**관련 법률 정리**

| 법률 | 내용 |
|------|------|
| **가정폭력 특례법 (1997년)** | 가해자를 무조건 전과자로 만들기보다 **'보호 처분' 우선** → 건강한 가정 회복에 주력 |
| **아동학대 특례법** | 가정 보호보다 **'아동 보호' 최우선** |
| **민법 제915조 (징계권) 폐지** | **2021년** 부모의 자녀 체벌(징계권) 조항 전면 폐지 |
| **스토킹 범죄 처벌 법률** | **2021년 시행**. 교제 폭력 전체를 포괄하는 특별법은 아님 |
| **삼진아웃제** | 3회 이상 적발 시 형사 처벌 |
| **1366 (여성긴급전화)** | 365일 24시간 지원. **남성 피해자도 동일하게 이용 가능** |
